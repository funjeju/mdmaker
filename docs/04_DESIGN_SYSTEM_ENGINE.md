# GLOBAL_DESIGN_SYSTEM.md

## 디자인 철학

본 서비스는 특정 페이지를 예쁘게 만드는 것이 목적이 아니다.

서비스 전체가

- 깔끔하고
- 여유롭고
- 직관적이며
- 전문적이고
- 신뢰감 있는

SaaS 제품처럼 느껴져야 한다.

사용자가 어떤 페이지를 방문하더라도

"잘 만들어진 서비스 같다"

라는 인상을 받는 것이 최우선 목표다.

# 서비스 전체 디자인 한줄 정의

이 서비스는

"노션 + 리니어 + 스트라이프 + 프레이머"

중간 어딘가에 위치한

현대적인 SaaS 스타일을 기준으로 한다.

모든 페이지와 컴포넌트는

카드 중심

넓은 여백

밝은 배경

부드러운 그림자

실제 서비스 UI 중심

이라는 원칙을 일관되게 유지해야 한다.
